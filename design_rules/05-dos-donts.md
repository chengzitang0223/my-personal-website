# 05-Dos & Don'ts: 苹果风绝对红线

## 🚫 绝对禁止 (Don'ts)
1.  **禁止使用高饱和度的杂乱色彩**：整个页面除了主按钮（蓝色）和状态色（红/绿），其他全都要克制在黑白灰之中。
2.  **禁止信息堆砌**：不要试图在一个视图里塞满所有字段。如果不重要，就隐藏到二级页面或折叠面板中。
3.  **禁止使用复杂的阴影**：绝不使用又黑又脏的 Drop Shadow。如果必须用，使用极大模糊半径、极低透明度的软阴影（如 `shadow-[0_8px_30px_rgb(0,0,0,0.04)]`）。
4.  **禁止使用非系统标准图标**：图标必须线条粗细一致，推荐使用类似于 SF Symbols 的极简线框图标。

## ✅ 必须遵守 (Dos)
1.  **留白即正义**：当你觉得元素之间太紧凑时，毫不犹豫地加大 `margin` 或 `padding`。
2.  **文本对齐**：大段说明性文字必须使用左对齐，坚决抵制大段文字居中对齐（除了极短的 Slogan）。
3.  **视觉层级**：永远确保用户第一眼只能看到最重要的那一个操作。
